## Getting Started

1. **Clone the repository**
   ```powershell
   git clone https://github.com/Mantelgen/Social-Network.git
   ```
2. **Build and run the project**
   ```powershell
   cd Social-Network
   ./gradlew run
   ```
   > On Windows, use `gradlew.bat run` instead.
